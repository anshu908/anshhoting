Quick Reference

🐍 Python Quick Start PYTHON
Create main.py with your app. Add requirements.txt for packages. Use os.environ['PORT'] to get your assigned port. Zip folder and upload.
💚 Node.js Quick Start NODE
Create index.js. Add package.json for npm deps. Use process.env.PORT for port. Zip and upload.
📦 ZIP Structure STRUCTURE
Your zip can contain a flat folder or a single top-level subfolder — we auto-flatten it. Entry file must be in root of the project (not in subfolders).
🔌 Port & URL NETWORK
Each project gets a unique port 5001–5200. The platform itself runs on port 5000. Access your app at http://localhost:PORT
⚡ Auto Deploy Flow DEPLOY
1. Upload file → 2. Pick entry file → 3. Deps auto-install → 4. Project auto-starts. Watch everything live in the terminal log viewer.
🛡 Security Notes SECURITY
Passwords hashed with SHA-256. File uploads restricted to safe types. Path traversal prevented. 200MB max upload. Session-based auth with 24h expiry.
