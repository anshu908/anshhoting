⎇ GitHub Repository
Deploy any public GitHub repo instantly
✅ What happens
1. 🔗 We git clone the repository
2. 🔍 Auto-detect language (Python/Node.js)
3. 📦 Install dependencies automatically
4. 🚀 Start your app on its assigned port
5. 📟 Watch it live in the terminal
📋 Supported URL formats
https://github.com/user/repo
https://github.com/user/repo.git
user/repo (shorthand)
⚠ Requirements
✓ Repository must be public
✓ Must have requirements.txt or package.json
✓ Entry file must be in root directory
✓ Use os.environ['PORT'] for port
